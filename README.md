# Tableau Portfolio

This repository contains a curated collection of Tableau dashboards designed to demonstrate business intelligence, data analysis, and data visualization skills across financial and operational domains.

Each project focuses on translating raw data into clear, decision-ready insights for business stakeholders.

## Purpose

This portfolio is designed to demonstrate the ability to:

- Build executive-level dashboards in Tableau
- Translate raw data into actionable insights
- Design KPI-driven business reporting systems
- Communicate findings clearly to stakeholders
- Organize analytics work in a professional portfolio format
